UKUPI Store

Es un proyecto desarrollado en el curso de Interación Hombre Maquina (IHM). Se trata de una simulación de una plataforma de comercio electrónico, diseñada para permitir a los usuarios explorar productos y añadirlos al carrito de compras. Es ideal para mostrar el flujo de navegación y la interacción del usuario con la tienda.

Medical Clinics - Historias Clínicas

Desarrollado con Spring Boot, HTML, CSS, JavaScript y PostgreSQL.

Ver Código Demo en Vivo

Medical Clinics es un proyecto desarrollado para el curso de Ingeniería de software II, el cual es un sistema de gestión de historias clínicas diseñado para facilitar el almacenamiento y la gestión de la información médica tanto para los médicos como para los pacientes. Los médicos pueden acceder y actualizar los historiales de los pacientes de forma segura, mientras que los pacientes pueden consultar su historial médico, agendar citas y recibir notificaciones sobre su salud.

Vet - Gestión Veterinaria

Desarrollado con Spring Boot y Angular 18.

Ver Código Demo en Vivo

Vet

Es una plataforma de gestión diseñada para veterinarias que permite a los propietarios de mascotas agendar citas, consultar historiales médicos, y recibir recordatorios de tratamientos y vacunas. El sistema facilita a los veterinarios mantener un registro detallado de la salud de cada animal y optimiza la programación de citas, mejorando la eficiencia y la atención a los pacientes animales. (El proyecto está en desarrollo).
